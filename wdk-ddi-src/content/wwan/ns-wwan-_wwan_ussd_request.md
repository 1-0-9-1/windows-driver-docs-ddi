# _WWAN_USSD_REQUEST structure


## -description


The WWAN_USSD_REQUEST structure describes an Unstructured Supplementary Service Data (USSD) request.


## -struct-fields




### -field RequestType

The type of USSD request.


### -field UssdString

The USSD string that accompanies the request.
