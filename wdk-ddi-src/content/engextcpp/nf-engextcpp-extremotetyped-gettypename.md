# ExtRemoteTyped::GetTypeName method


## -description


The <b>GetTypeName</b> method returns the type name of the typed data represented by this object.


## -syntax


````
PSTR GetTypeName();
````


## -parameters






## -returns



<b>GetTypeName</b> returns a string that contains the name of the type.
