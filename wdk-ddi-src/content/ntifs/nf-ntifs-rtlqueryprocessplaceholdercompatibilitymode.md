# RtlQueryProcessPlaceholderCompatibilityMode function


## -description

The **RtlSetProcessPlaceholderCompatibilityMode** function returns the placeholder compatibility mode for the current process.

## -parameters


## -returns
This function returns  the process's placeholder compatibily mode (PHCM_xxx), or a negative value on error (PCHM_ERROR_xxx).
